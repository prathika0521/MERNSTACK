What Makes It Responsive?

Three core CSS concepts that make this page work on any screen size.

↔

Flexbox Direction

flex-direction: row places items side by side on desktop. On mobile, @media switches it to column so they stack vertically.

📐

Flex Wrap

flex-wrap: wrap lets items automatically drop to the next line when there isn't enough horizontal space — like the feature cards below.

📱

Media Queries

@media (max-width: 768px) applies different CSS rules only on smaller screens — changing layout, font sizes, and spacing for mobile.
